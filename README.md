# API Testing Presentation

Reveal.js slides for the Node.js API Integration Testing Framework presentation.

## View Slides

Open `slides/index.html` in a browser, or run a local server:

```bash
cd api-testing-slides
python3 -m http.server 8000
# Open http://localhost:8000/slides/
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `→` / `Space` | Next slide |
| `←` | Previous slide |
| `Esc` | Overview mode |
| `F` | Fullscreen |
| `S` | Speaker notes |
